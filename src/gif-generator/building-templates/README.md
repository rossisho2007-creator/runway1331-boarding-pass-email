## GIF Specifications
- Dimensions: 580x400px
- Duration: 3-5 seconds
- File size: Under 1MB
- Colors: Max 256 (GIF limitation)
- Frame rate: 10-15 fps

## Animation Sequence
1. Frame 1: Full property map with reception AND zone clearly marked (static fallback for Outlook)
2. Frames 2-30: Dashed line animates from reception to the zone
3. Frames 31-45: Zone pulses/highlights
4. Frames 46-60: Hold final frame with You are here marker

## Email Display
Guest receives zone GIF showing path to their 2-building area.
Email text specifies exact building number (e.g., Building 84).

# Building GIF Templates

Place your animated map GIFs here.

## Naming Convention
- building-a.gif - Path to Building A
- building-b.gif - Path to Building B
- building-c.gif - Path to Building C
- building-default.gif - Fallback for any building

## GIF Specifications
- Dimensions: 580x400px
- Duration: 3-5 seconds
- File size: Under 1MB
- Colors: Max 256 (GIF limitation)
- Frame rate: 10-15 fps

## Animation Sequence
1. Frame 1-15: Show full property map with reception marked
2. Frame 16-30: Animate dashed line from reception to building
3. Frame 31-45: Pulse/highlight the destination building
4. Frame 46-60: Hold final frame with You are here marker

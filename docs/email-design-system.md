# Email Design System

## Aviation Metaphor Elements
The entire email uses a "Boarding Pass" metaphor to make guests feel like they're embarking on a journey.

### Key Aviation Terms Used
- **Boarding Pass** = The email itself
- **Gate** = Building block identifier
- **Squawk Code** = Door access PIN
- **Flight** = The guest's stay
- **Crew** = Runway1331 staff
- **Arrival** = Check-in
